# Grok Build agent instructions

Follow the root **[AGENTS.md](../AGENTS.md)** for all project rules.

## Grok priorities

1. Keep the light, sober visual system (warm paper + ink navy).
2. **Dose visuals:** charts, SVG, motion, and diagrams only when they help understand **this** lesson — not on every page. Prefer one primary demo. See root `AGENTS.md` → “Visual media: dose, do not decorate”.
3. Apply new design rules to site tokens/chrome when teaching them.
4. Write lesson rules in STE100 (`docs/STE100.md`).
5. Never reproduce paid-course content; create original examples only.
6. Keep everything componentized (`ui` / `lab` / `lessons`) so the site transforms with each technique.

## Common tasks

- New lesson page under `/lab/[slug]`
- Expand design tokens in `globals.css`
- Improve bad/good comparison demos
- Add diagrams or charts that clarify a single rule
